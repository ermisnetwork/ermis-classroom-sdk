# Getting Started

Welcome to the Ermis Classroom SDK! This section will help you get up and running quickly.

## Prerequisites

- **Node.js** 18+ (LTS recommended)
- **npm**, **pnpm**, or **yarn**
- Modern browser with WebCodecs support (Chrome 94+, Safari 15.4+, Edge 94+)

## What You'll Learn

1. [Installation](installation.md) - How to install the SDK
2. [Quick Start](quick-start.md) - Build your first video call in minutes
3. [FAQ](faq.md) - Common questions and answers

## Browser Support

| Browser | WebTransport | WebRTC | WebCodecs |
|---------|--------------|--------|-----------|
| Chrome 94+ | ✅ | ✅ | ✅ |
| Edge 94+ | ✅ | ✅ | ✅ |
| Safari 15.4+ | ❌ | ✅ | ✅ |
| Firefox | ❌ | ✅ | ⚠️ Partial |

> [!NOTE]
> The SDK automatically falls back to WebRTC when WebTransport is not available.
