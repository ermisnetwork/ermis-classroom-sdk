# Features

This section covers advanced features of the SDK.

## Available Features

- [Screen Share](screen-share.md) - Share your screen with participants
- [Device Switching](device-switching.md) - Switch cameras and microphones
- [Breakout Rooms](breakout-rooms.md) - Create sub-rooms for group discussions
- [Pin for Everyone](pin-for-everyone.md) - Pin a participant's video/screen share for all attendees
